# Azure Blob Storgage with Asp.Net MVC

This is Asp.Net MVC project for uploading image on Azure Blob Storage.

# Prerequisities

Visual Studio 2015
SQL Server 2012


# Deployment

Download the code.
Change the web.config connection string.
Create Azure blob storage and change the storage key.
Run the project.
